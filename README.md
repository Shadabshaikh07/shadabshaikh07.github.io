# Expense Tracker

A simple, client-side expense tracking web application built with HTML, CSS, and JavaScript.

The app allows users to track daily expenses, categorize them, view monthly summaries, and export data to CSV â€” all without a backend.

## Features

- Add expenses with:
  - Name
  - Category
  - Date
  - Amount (PLN)
- Automatic total calculation
- Category-wise totals
- Monthly expense summaries
- CSV export for spreadsheets
- Persistent storage using browser localStorage
- Responsive, centered layout
